# awww-bin (An animated wallpaper daemon for wayland)

Compiled binary distribution of awww.

Upstream project: https://codeberg.org/LGFae/awww

## Download

Download the latest release from the [Releases](https://github.com/IncogCyberpunk/awww-bin/releases) section.

## Installation

```bash
tar -xzf awww-*-x86_64.tar.gz
sudo cp -r usr/* /usr/
```

## Contents

- `usr/bin/awww` - Main awww binary
- `usr/bin/awww-daemon` - Daemon binary
- `usr/share/man/man1/` - Manual pages
