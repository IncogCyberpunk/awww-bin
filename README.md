# awww-bin (An animated wallpaper daemon for wayland, controlled at runtime)

Compiled binary distribution of awww.

Upstream project: https://codeberg.org/LGFae/awww

## Download

[awww-0.11.2-x86_64.tar.gz](https://github.com/IncogCyberpunk/awww-bin/raw/main/awww-0.11.2-x86_64.tar.gz)

## Installation

```bash
tar -xzf awww-0.11.2-x86_64.tar.gz
sudo cp -r usr/* /usr/
```

## Contents

- `usr/bin/awww` - Main awww binary
- `usr/bin/awww-daemon` - Daemon binary
- `usr/share/man/man1/` - Manual pages
